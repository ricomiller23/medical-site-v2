# Turbo Mode Workflow

// turbo-all

This workflow enables automatic approval for safe commands in the medical-site-v2 project.

## Usage
When working on this project, the agent will automatically run safe commands without asking for confirmation.

## Safe Operations
- Git operations (add, commit, push)
- npm/node commands
- File verification commands (curl, head, tail, cat, grep)
- Build and deploy commands
